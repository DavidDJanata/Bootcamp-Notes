OOP - Object Oriented Programming

  • A programming concept focues around objects
  • Problems are solved by using collections of objects that work together
  • There are five core concepts of OOP:
    1. Inheritance: New objects created based on other objects
    2. Compostion: Objects containing other objects
    3. Polymorphism: Multiple object types implement the same functionality
    4. Encapsulation: Object data (and functions) are neatly stored
    5. Abstraction: Create a simple model of something complex
  • We use objects to model real world things in code and provide functionality that would otherwise be hard or impossible

TypeScript

  • TypeScript is a typed superset of JavaScript, adding new rules that dictate how different values areused
  • TypeScript is a compiler that converts TypeScript syntac into clean JavaScript that can be consumed anywhere
  • TypeScript is a language server that provides instant error checking in your IDE while you code
  • TypeScript checks for consistency with data types when assigning values
  • Browsers do not understand TypeScript, therefore it is converted using tsc (TypeScript Compiler)
  • TSC uses transpiling, which takes unsupported syntex and converts it into clean supported syntax, based on the set target environment (ES5, ES6, ES2020, etc) and then compiles it to run
  • Does not enforce OOP principles, but allows us to implement OOP concepts into our code in ways that JS alone can't
