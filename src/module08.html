Installing / Running TypeScript

You will need to have Node.js installed and then run the following npm command:
npm i typescript --save-dev
or
npm i -D typescript
This will install typescript as a developer dependency instead of a standard dependency.

0801 TS Setup

  • In package.json, the scripts are npm commands that can be run. The left side being the commands, and the right side being the instructions.
  • npm (Node Package Manager): Used for install, uninstalls, running programs
  • npx (Node Package Executor): Runs what ever command is immediatley after the commmand
  • npm build: Runs npx tsc (TypeScript Compiler)
  • npm run start: Runs npm build, and then runs command "node dist/index.js" - which creates a folder named "dist" and stores the newly transpiled "index.js" file in it
  • Just like how package.json contains our js settings, tsconfig.json contains our ts settings
  • In tsconfig.json:
    • target: sets the intended javascript (ES) version
    • module: sets which module code is generated
    • rootDir: sets the root folder or your source files
    • outDir: sets the output folder to send files to
    • noEmitOnError: determines whether or not the tsc will continue if errors are detected
    • include: makes sure all associated files are included

0802 TS Setup

  • Install Typescript as a developer dependency via npm i -D typescript
  • Bring tsconfig.json file from 0801 to reuse code
  • Or: npx tsc --init (Not reccomended early on)
