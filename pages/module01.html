Git Commands

ls (list)

Lists all files and paths in current directory.

pwd (print working directory)

Displays the current directory / file path to current location.

cd (change directory)

Moves into a new directory based on input. cd filename or cd filepath

cd ..

Moves up one directory in current path.

mkdir (make directory)
